It's a simple Python script whose main purpose is to convert audios to different formats (like .wav to .mp3). You can choose to convert a single audio file or an entire folder.

***Technologies used:*** 
**Language:** Python 
**Library:** Pydub

***How to use:*** 
Clone this repository, then, install the library Pydub with 'pip install pydub' and execute the script in the terminal
